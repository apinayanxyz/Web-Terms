    Term Definitions
    Agile Methodology Agile is a type of software development project management method, it helps to break down large projects into more manageable sized loads so you can get feedback after each stage of development, this development methodology helps to get feedback at every stage instead of having to change large amounts of work after having a whole project draft and instead having chunks being developed and having feedback off those. ()
    AJAX Ajax stands for Asynchronous JavaScript and Xml. It is used for creating better, efficient, faster and more interactive web applications alongside other programming languages like XML, HTML,CSS and Java script. It is a programming concept, not new technology. AJAX allows web applications to be updated asynchronously and send data and retrieve data from a background server, allowing a web page to be continuously updated without the need of refreshing the website.     CSS CSS stands for Cascading Style Sheets. HTML allows the user to add paragraphs , headlines , embeds images and videos to the site while CSS can allow you to change the page layout , colours and fonts that fits the aesthetic. A CSS rule is made up of one or more selectors and a semicolon-separated list of declarations inside brackets. Each declaration has a property and the value separated by a colon. A rule is made up of one or more selectors. In the example below, the selector is a paragraph element represented by the letter P. This means that the rule will apply to all paragraphs. This declaration states that the text colour inside a paragraph will be teal. Note the spelling of “color” (codeinstitute, What is CSS? The Ultimate Intro, 2022)
    Cloud Cloud is being able to access documents , emails and database through the internet. It allows users to access the same document at the same time on any devices . For example , Google Drive is a cloud based storage service that allows users to access files and store online. The service can sync documents from any devices (cloudflare, 2022)
    Web Accessibility Web accessibility refers to the accessibility of the Web for people with disabilities. In more specific terms, Web accessibility means that people with disabilities can perceive, understand, navigate, and interact with the Web, and also, they can contribute to the Web. Other groups, such as older people with changing abilities as they age, also benefit from web accessibility. (Ali, 2022)
    Wiki The term wiki refers to a website or online resource that can be edited by multiple users. Wikipedia, for example, is a publicly accessible wiki. Organizations use others to manage information internally,enabling teams to collaborate more effectively and share knowledge easily. (Team, 2022)
    W3C A W3C (World Wide Web Consortium) is an international organization that develops Web standards. WC3 sets and promotes web-based standards with the goal of improving the web. (Gills, 2022))
    WinSCP WinSCP is a program for windows whose main function is to allow a user to transfer files between a local and remote computer. The program also provides a GUI, an integrated text editor, scripting, a basic file manager and other features. ((WinSCP), 2017)
    XML XML (eXtensible Markup Language) is a programming language, like HTML,that’s used to store and carry data. XML is used in tandem with HTML, with the HTML being used show the data stored by the XML, and allows for data to be stored in separate files from the HTML code but, unlike HTML,HTML does not use predefined tags and instead uses tags that are first defined by the author. (w3schools,2022)

    XSL XSL is a stylesheet language, like CSS, for XML that’s used to tell browsers how XML elements should be displayed. XSL consists of 3 main parts: XSLT, a language for transforming XML into another document, or for manipulating attributes and elements; XPath, a language for navigating elements and attributes in an XML documents; and XQuery, a language for querying XML. (w3schools, XSLT Introduction,2022) (w3schools, XSL(T) Languages,2022)
